Como ejecutar la herramienta
01. Ejecute el script de sql, el cual se encuentra en la carpeta "ScriptBD" en un motor de bases de datos de SQL Server.
02. Abra la solucion en visual studio (preferiblemente el 2017).
03. Establasca como proyecto de inicio el proyecto por nombre "FrameworkAnaliticaVisual".
04. Ejecute la solución y establesca lo que se le solicita en el wizard.
05. En el segundo paso, establesca las tablas de su preferencia para generar las vistas.
06. En el tercer paso, seleccione los campos de las columna con los campos de las vistas que desea asociar.
07. Al establecer el directorio, seleccione el directorio raiz de la solucion, acontinuacion seleccione el archivo en la siguiente ubicación "\CanvasViews\Models\clsVista.cs".
08. Una vez seleccionado, finalize con el wizard, el cual le indicara ssi la clase se ha escrito correctamente.
09. Detenga la ejecución del programa, a continuacion, establesca el proyecto "CanvasViews" como proyecto de inicio.
10. Por ultimo ejecute el proyecto, y este le mostra las vizualizaciones con los datos correspondientes a la selección previa.


NOTA: en el primer paso de establecer la conexion a base de datos, introdusca los credenciales de la base de datos anteriomente creada.
